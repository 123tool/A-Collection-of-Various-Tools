**Condition:**  
- Windows 7 or later  
- .NET Framework 4.7.2 or above  
- [Java 8](https://java.com/en/download/) or [JDK 8](https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html) , it doesn't matter which
  

**Feature:**  
**Decompile/Compile**  
Indonesian: Decomposition and compilation of APK, DEX and JAR files, with space and symbol support. Characters like Japanese, Chinese.. are not supported. Can be canceled during operation  
  
**Sign/Zipalign**  
Signing and ZipAligning APK/JAR files. Can be canceled during operation
  

**Extract/Zip APK**  
Extract and zip APK with 7z Compression level 0-9. Can be canceled during operation  
  
**APK Information**  
Show APK information with an icon using the aapt dump badge and a link to the Play Store  
  
**Framework**  
Install your framework and manage framework paths  
  
**Log output**  
Look at your logs for errors.
  

**Option**  
Change tool options, decompile, compile, sign and zipalign. Options: Java stack, Apktool version, apksigner version, directory, Java path, Windows Explorer integration, remember window position and more  
All options and text boxes are saved as config.xml and will be loaded on launch  
Quick option in main.  
  
**Drag and drop**  
Place the APK file, DEX, JAR, and Decompile folder on the button to perform the action. English: Drop outer button to select/open as file/folder  
  
**Apktool.jar version selection**  
Can switch to other version of apktool.jar in option  
Add another apktool.jar in "Apktool" folder  
  
**Shortcut**  
4 buttons to open working directory of decompiled APK, compiled APK, extracted APK and zipped APK  
Jumplist shortcut  
  
**Windows Explorer Integration**  
Perform actions by simply right-clicking on the APK file and selecting the following options, decompose or compile APK, DEX, and JAR files, Zipalign APK, Check alignment, Sign APK and Install APK
